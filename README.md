## Getting Started

Activate corepack:

```bash
corepack enable
```

Install dependencies:

```bash
yarn install
```

Open [http://localhost:3000/](http://localhost:3000/). Pages will live-refresh when source files are changed.

> Note: remember to set your environment variables in a `.env.local` file.
